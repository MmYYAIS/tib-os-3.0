

## ✨ 项目简介

**TIB OS v3.0 (The Indie Tech Investor Operating System)** 是一个将**十年投资经验与思考“代码化”**的 AI 投资辅助系统。它将复杂的投资决策流程拆解为 **15 个核心投资元技能 (Prompts)**，并整合到一个**开箱即用的前端交互界面**中。旨在帮助独立投资者告别繁琐的手动输入，实现更理性、更高效、更系统化的投资决策。

我们相信，投资的价值在于深度思考和逻辑博弈，而非重复性的数据录入。TIB OS v3.0 将 AI 深度融入投资全流程，让你的投资决策拥有“SaaS 级”体验。

## 💎 核心特性

*   **15 个核心投资元技能**：涵盖从市场扫描、深度调研、交易执行到持仓管理与复盘的投资全流程。
*   **开箱即用的交互界面**：一个独立的 `index.html` 文件，双击即可在浏览器中运行，无需任何环境配置。
*   **智能变量配置**：自动识别 Prompt 模板中的变量，生成直观的输入框，告别手动修改 Prompt 的繁琐。
*   **本地缓存记忆**：利用浏览器 `localStorage` 自动保存用户输入，下次打开页面数据不丢失，实现无缝工作流。
*   **自定义模板与编辑**：支持在浏览器中直接编辑和保存 Prompt 模板，打造完全符合个人投资习惯的专属工具。
*   **轻量化与可扩展性**：纯 HTML/CSS/JavaScript 实现，不依赖外部库，易于理解、修改和扩展。
*   **一键魔改**：对于开发者，可轻松克隆项目，基于豆包应用开发平台或本地环境进行深度定制。
